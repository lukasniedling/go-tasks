# Übungsaufgaben zur Vertiefung der Programmierkenntnisse in Go

In diesem Repository finden sich verschiedene Übungsaufgaben, die dazu
dienen sollen, die Programmierkenntnisse in der Programmiersprache Go zu vertiefen.

Es gibt mehrere Packages mit Aufgaben, die jeweils zusammengehören und aufeinander
aufbauen bzw. zusammen bearbeitet werden sollten.
Es geht u.A. darum, Techniken und Lösungsansätze in ähnlichen Aufgabenstellungen
wiederzuverwenden und zu erkennen, wie man allgemeine Lösungen für ähnliche
Probleme formulieren kann.
